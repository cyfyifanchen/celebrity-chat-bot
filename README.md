## Tech stack
* Auth, Clerk 
* Next 13, App Router 
* Prisma, schema control
* Plantscale, DB
* Vercel, Deployment
* Tailwind shadCN, styling 
* Pinecone, Memory Service
* OpenAI, Chat AI response
* Cloudiary, media service
* Strip, Payment(unfinished)
